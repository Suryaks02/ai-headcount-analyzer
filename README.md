# 🤖 AI Headcount & Capacity Analyzer

A professional full-stack application using **TensorFlow.js** to monitor room occupancy in real-time. Featuring a sleek Neon-Cyberpunk UI and a robust Node.js backend.

## 🚀 Key Features
* **👥 Real-time Headcount:** Live person detection via webcam.
* **🚫 Capacity Guard:** Visual and logic-based alerts for overcrowding.
* **🖼️ Image Analysis:** Drag-and-drop photo uploads for instant person counting.
* **📊 Data Logging:** Automated backend history storage in JSON.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Neon-Grid), JavaScript
* **AI Model:** COCO-SSD (TensorFlow.js)
* **Backend:** Node.js, Express.js
* **File Handling:** Multer

## ⚙️ Installation & Setup
1. Clone the repo: `git clone https://github.com/Suryaks02/ai-headcount-analyzer.git`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
4. Open `http://localhost:3000` in your browser.
